## Uruchamianie projektu
### Virtual environment (dependencies)
>##### Linux, Mac 
```bash
source venv/bin/activate
```
>##### Windows
```powershell
set-executionpolicy remotesigned
.\venv-windows\Scripts\Activate.ps1
```
### Local server
```bash
python manage.py runserver
```
